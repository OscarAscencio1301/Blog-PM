## Frontend
1. Accede a la carpeta blog-frontend
```bash
 cd blog-frontend/
```
2. Crea el archivo `.env` en base a `.env.example`.
  ```bash
  VITE_URL_BACKEND=URL_SERVER_BACKEND
  ```
2. El VITE_URL_BACKEND lo obtienes al ejecutar blog-backend.
